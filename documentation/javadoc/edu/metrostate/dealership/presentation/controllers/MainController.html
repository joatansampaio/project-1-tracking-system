Package edu.metrostate.dealership.presentation.controllers

Class MainController

  • All Implemented Interfaces:
    
public final class MainController

    Main controller for the dealership management application. Manages the primary UI with both vehicle and dealer tables, navigation between views, and actions such as import/export, adding/deleting vehicles or dealers, and toggling properties.

Package edu.metrostate.dealership.infrastructure.database

Class Database

  • All Implemented Interfaces:
    
public final class Database

    In-memory singleton database implementation for the dealership management system. Stores all dealers and vehicles data and provides CRUD operations for them. This class follows the singleton pattern for application-wide access to a single data source.
